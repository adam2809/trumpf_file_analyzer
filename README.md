### Instrukcja

Żeby zbudować projekt i wykonać testy wywołaj skrypt `build_and_run` z korzenia repozytorium. Plik wykonywalny znajdziesz w `bin` pod nazwą `trumpf_zadanie_rekrutacyjne`. Żeby skorzystać z programu podaj ścieżkę pliku wejściowego jako pierwszy argument, ścieżkę pliku wyjściowego jako drugi argument i wielkość okna jako trzeci.
